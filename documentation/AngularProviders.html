AngularProviders($http)

SPGlideAjax

Constructor

new AngularProviders($http)

A simplified version of GlideAJAX for use within the Service Portal. This specifically implements getAnswer and getXMLAnswer with some modernization for Promise support through Angular's $http service.

Example
Example Script Include Declaration:
```javascript
var Example = Class.create();
Example.prototype = Object.extendsObject(AbstractAjaxProcessor, {
	"type": "Example",
	"functionA": function() {
		return this.getParameter("a");
	}
});
```

Example usage of SPGlideAjax:
(This will print "9" as a warning to the browser's console)
```javascript
 function _controller($scope, SPGlideAjax) {
 	var ga = new SPGlideAjax("Example");
 	ga.addParam("sysparm_name", "functionA");
 	ga.addParam("a", 9);
 	ga.getXMLAnswer(console.warn);
 }
```
