Contains assistive methods for working with the ActionTable widget and acts as an anchor point for leveraging server-side data with script includes while not using the SPGlideAJAX functionality. Note that ActionTable supports pulling data from a client callable script include as well as leveraging the REST API (Though authentication may pose issues with REST). Extending this class instead of AbstractAjaxProcessor is not necessary from a functionality perspective, but does help to track or imply why that script exists.
